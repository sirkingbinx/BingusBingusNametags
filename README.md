# BingusBingusNametags
> [!NOTE]
> This isn't part of BingusNametags, this is my personal plugin that i use with it. Get the mod at https://github.com/sirkingbinx/BingusNametags

## Tags
The tags are organized like this. (note: all text encased in angle brackets `<>` are comments, not displayed on the tags)
```
  ██ 000    ABC123456789
<clr code>   <player id>
         JOEMAMA
     < player name >
         SteamVR
       < platform >
   [Mod1] [Mod2] [Mod3]
< list of mods (prop checker) >
```
